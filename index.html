OpenCore ExposeSensitiveData Generator



Select the policies you want enabled from the list below:

0x01 (bit 0) - 0x01, expose the printable booter path as a UEFI variable.
0x02 (bit 1) - 0x02, expose the OpenCore version as a UEFI variable.
0x04 (bit 2) - 0x04, expose the OpenCore version in the OpenCore picker menu title.
0x08 (bit 3) - 0x08, expose OEM information as a set of UEFI variables.
